## Window/Pane/Tab management

Ctrl+W...

**Focus**
- h,j,k,l -- goto split
- w -- walk splits

**Resizing**
- < -- decrease width
- > -- increase width
- + -- increase height
- - -- decrease height

- | -- maximize vertical
- _ -- maximize horizontal
- = -- equalize all splits

**Moving**
- H,J,K,L -- filling half of the window
- T -- to a new tab
