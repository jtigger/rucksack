## Manual Configuration Tweaks

- System Preferences:
  - Trackpad
    - Point & Click:
      - Secondary click: bottom-right corner
    - More Gestures
      - Swipe between pages: Off
      - Swipe between full screen applications: 3-fingers + left/rigt
      - Mission Control: 3-fingers + up
      - App Expose: 3-fingers + down
      - Notification Center: On

  - Keyboard > Shortcuts 
    - Mission Control
        - Mission Control = [Shift+Crtl+Opt+Cmd]+Up
        - Application windows = [Shift+Crtl+Opt+Cmd]+Down
        - Notification Center = [Shift+Ctrl+Opt+Cmd]+N
        - Show Desktop = [Shift+Ctrl+Opt+Cmd]+D
        - Move left a space = [Shift+Ctrl+Opt+Cmd]+[
        - Move right a space = [Shift+Ctrl+Opt+Cmd]+]
